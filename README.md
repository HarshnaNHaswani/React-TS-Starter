# React TS Starter using Webpack 5

Features:
- Separate development and production configuration
- TypeScript
- React
- React-Router
- Hot Reload
- Lint
- Lint Time
- Prettier
- Cross-Env
- Lazy Loading
- Error Boundary setup
- Bundle Analyzer
- Automatic asset copy to build folder
- Basic Folder Structure setup 

Support:
- Latest ES Modules
- XML
- MarkDown
- TOML
- TXT
- CSS
- PDF
- Images
- Font Files 
- Modules
- YAML 
- JSON
